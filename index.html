What Is MEV?

Maximally Extractable Value or MEV is the revenue the block producer (e.g. validator, miner, sequencer) can extract via front running, injecting, reordering or censoring transactions. The MEV extracted in 2020 alone was worth more than $314M — and that is only a lower bound.

How Shutter works

The Shutter Network uses a threshold cryptography-based distributed key generation (DKG) protocol.

The mechanism used by Shutter can be implemented into various layers. The first instantiation we built is a simple drop-in solution for smart contracts at the application level, which is gas efficient and doesn’t require L1/L2 protocol changes.
